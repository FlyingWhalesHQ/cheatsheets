## Syntaxs

### Headers

```
# h1
## h2
### h3
#### h4
##### h5
###### h6
```

Header 1 and header 2 could also use the following markup

```
Header 1
========

Header 2
--------
```

### Emphasis

```
*italic*
_italic_

**bold**
__bold__

***bold italic***
___bold italic___

~~strikethrough~~

`code`
```
